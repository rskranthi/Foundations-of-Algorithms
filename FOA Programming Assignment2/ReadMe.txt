1. Download the zip file name FOA Programming Assignment2.zip onto your local machine.
2.  Import the project into existing workspace by selecting archive file.
3.Goto Project folder--> Main-->Run as--> Run configurations-->Arguments and give the path of the input file as the program argument.
4.Run the program
5. Select the input choice from the options given.
6. After every input is taken, corresponding match schedule is printed and the program waits for next input type until you exit it.
7. For all values >10, the text file named "output.txt" is appended or created and placed in the project workspace.
